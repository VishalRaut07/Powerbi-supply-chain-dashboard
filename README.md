# 🚀 Power BI Supply Chain & Logistics Dashboard

## 📌 Project Overview
This project focuses on analyzing supply chain operations including supplier performance, inventory management, shipment tracking, and revenue insights using Power BI.

The goal is to transform raw data into interactive dashboards for better business decision-making.

---

## 📊 Key Insights

- Identified suppliers with the best lead time performance
- Analyzed inventory stock across multiple product categories
- Detected shipment delays by different logistics carriers
- Compared revenue contribution across sales platforms
- Evaluated KPIs like Revenue, Profit, Profit Margin, and Order Quantity
